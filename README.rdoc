= Deception!

A turn-based strategy game engine. Basis for Twitter Werewolf game (work in progress).

== Usage

It's a Rails app, dummy. Copy / customize your <tt>database.yml</tt> and <tt>settings.yml</tt> from the provided examples in the config directory. Then run <tt>rake db:migrate</tt>. A sample game script is provided. See the integration tests for example.
